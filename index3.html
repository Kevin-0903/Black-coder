Welcome to Smart Traffic AI

A simple and intelligent traffic management system using cameras and AI to detect vehicles, optimize signals, and automatically catch violations.

About the Project

This project uses AI-based vehicle detection to manage traffic signals dynamically. It helps reduce congestion, improve traffic flow, and automate violation detection. This is the simple website structure used at the beginning of the documentation.

Features

  • Live camera monitoring
  • Vehicle detection using YOLO
  • Smart signal timing
  • Violation detection
  • Dashboard for traffic police
